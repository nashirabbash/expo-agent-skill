# 📂 others/

Miscellaneous uncategorized modules.

## Available Modules

- **`CLAUDE.md`** (CLAUDE.md)
- **`captureRef.md`** (react-native-view-shot) - A library that allows you to capture a React Native view and save it as an image.
- **`gesture-handler.md`** (react-native-gesture-handler) - A library that provides an API for handling complex gestures.
- **`keyboard-controller.md`** (react-native-keyboard-controller) - A library that provides a Keyboard manager that works in an identical way on Android and iOS
- **`light-sensor.md`** (LightSensor) - A library that provides access to the device's light sensor.
- **`linear-gradient.md`** (LinearGradient) - A universal React component that renders a gradient view.
- **`linking.md`** (Linking) - An API that provides methods to create and open deep links universally.
- **`maps.md`** (Maps) - A library that provides access to Google Maps on Android and Apple Maps on iOS.
- **`media-library.md`** (MediaLibrary) - A library that provides access to the device's media library.
- **`mesh-gradient.md`** (MeshGradient) - A module that exposes MeshGradient view from SwiftUI to React Native.
- **`navigation-bar.md`** (NavigationBar) - A library that provides access to various interactions with the native navigation bar on Android.
- **`screen-capture.md`** (ScreenCapture) - A library that allows you to protect screens in your app from being captured or recorded.
- **`screen-orientation.md`** (ScreenOrientation) - A universal library for managing a device's screen orientation.
- **`status-bar.md`** (StatusBar) - A library that provides the same interface as the React Native StatusBar API, but with slightly different defaults to work great in Expo environments.
- **`stripe.md`** ('@stripe/stripe-react-native') - A library that provides access to native APIs for integrating Stripe payments.
- **`system-ui.md`** (SystemUI) - A library that allows interacting with system UI elements.
- **`third-party-overview.md`** (Third-party libraries supported in Expo Go) - A set of third-party libraries which support for is included by default in Expo Go environment.
- **`view-pager.md`** (react-native-pager-view) - A component library that provides a carousel-like view to swipe through pages of content.
- **`webbrowser.md`** (WebBrowser) - A library that provides access to the system's web browser and supports handling redirects.
