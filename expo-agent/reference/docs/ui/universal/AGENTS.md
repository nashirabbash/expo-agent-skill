# 📂 universal/

Universal UI components for all platforms.

## Available Modules

- **`bottomsheet.md`** (BottomSheet) - A modal sheet that slides up from the bottom of the screen.
- **`button.md`** (Button) - A pressable button with multiple visual variants.
- **`checkbox.md`** (Checkbox) - A toggle control that represents a checked or unchecked state.
- **`collapsible.md`** (Collapsible) - A labelled tappable header that toggles visibility of its content.
- **`column.md`** (Column) - A vertical layout container for universal @expo/ui components.
- **`fieldgroup.md`** (FieldGroup) - A scrollable container of grouped settings-style rows.
- **`host.md`** (Host) - A cross-platform Host component that wraps universal @expo/ui content.
- **`icon.md`** (Icon) - A platform-native icon — SF Symbol on iOS, Material Symbol on Android.
- **`index.md`** (Universal) - Cross-platform components for building shared UIs across Android, iOS, and web with @expo/ui.
- **`list.md`** (List) - A virtualized vertical container of rows, paired with a tappable ListItem primitive.
- **`picker.md`** (Picker) - A single-selection input with menu and wheel appearances.
- **`rnhostview.md`** (RNHostView) - A cross-platform component for hosting React Native views inside @expo/ui views.
- **`row.md`** (Row) - A horizontal layout container for universal @expo/ui components.
- **`scrollview.md`** (ScrollView) - A scrollable container that supports vertical or horizontal scrolling.
- **`slider.md`** (Slider) - A control for selecting a value from a continuous or stepped range.
- **`spacer.md`** (Spacer) - A layout spacer that produces empty space between siblings.
- **`switch.md`** (Switch) - A toggle control that switches between on and off states.
- **`text.md`** (Text) - A component for displaying styled text content.
- **`textinput.md`** (TextInput) - A text input backed by native SwiftUI and Jetpack Compose components, with a React Native-compatible API.
- **`universal.md`** (Universal) - Cross-platform components for building shared UIs across Android, iOS, and web with @expo/ui.
