# 📂 system/

Core system behaviors (Filesystem, Updates, TaskManager, AppState, etc).

## Available Modules

- **`app-integrity.md`** (AppIntegrity) - A library that provides access to Google's Play Integrity API on Android and Apple's App Attest service on iOS.
- **`application.md`** (Application) - A universal library that provides information about the native application's ID, app name, and build version at runtime.
- **`background-fetch.md`** (BackgroundFetch) - A library that provides API for performing background fetch tasks.
- **`background-task.md`** (BackgroundTask) - A library that provides an API for running background tasks.
- **`brownfield.md`** (Brownfield) - Toolkit and APIs for integrating Expo into existing native applications.
- **`build-properties.md`** (BuildProperties) - A config plugin that allows customizing native build properties during prebuild.
- **`clipboard.md`** (Clipboard) - A universal library that allows getting and setting Clipboard content.
- **`constants.md`** (Constants) - An API that provides system information that remains constant throughout the lifetime of your app's installation.
- **`crypto.md`** (Crypto) - A universal library for crypto operations.
- **`dev-client.md`** (DevClient) - A library that allows creating a development build and includes useful development tools.
- **`dev-menu.md`** (DevMenu) - A library that provides a developer menu for debug builds.
- **`expo.md`** (Expo) - Set of common methods and types for Expo and related packages.
- **`filesystem-legacy.md`** (FileSystem (legacy)) - A library that provides access to the local file system on the device.
- **`filesystem.md`** (Expo Docs Page) - Expo official documentation
- **`intent-launcher.md`** (IntentLauncher) - A library that provides an API to launch Android intents.
- **`keep-awake.md`** (KeepAwake) - A React component that prevents the screen from sleeping when rendered.
- **`localization.md`** (Localization) - A library that provides an interface for native user localization information.
- **`manifests.md`** (Manifests) - A library that provides types for Expo Manifests.
- **`server.md`** (Server) - Server-side API and runtime for Expo Router projects.
- **`symbols.md`** (Symbols) - A library that allows access to native symbols.
- **`task-manager.md`** (TaskManager) - A library that provides support for tasks that can run in the background.
- **`updates.md`** (Updates) - A library that enables your app to manage remote updates to your application code.
- **`widgets.md`** (Widgets) - A library to build iOS home screen widgets and Live Activities using Expo UI components.
