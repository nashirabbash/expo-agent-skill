# 📂 services/

System services and integrations (Auth, Contacts, Calendar, Speech, etc).

## Available Modules

- **`age-range.md`** (AgeRange) - A library that provides access to age range information using Play Age Signals API on Android and Declared Age Range framework on iOS.
- **`apple-authentication.md`** (AppleAuthentication) - A library that provides Sign-in with Apple capability for iOS.
- **`auth-session.md`** (AuthSession) - A universal library that provides an API to handle browser-based authentication.
- **`calendar-legacy.md`** (Calendar (legacy)) - A library that provides an API for interacting with the device's system calendars, events, reminders, and associated records.
- **`calendar.md`** (Calendar) - A library that provides an API for interacting with the device's system calendars, events, reminders, and associated records.
- **`contacts-legacy.md`** (Contacts (legacy)) - A library that provides access to the phone's system contacts.
- **`contacts.md`** (Contacts) - A library that provides access to the phone's system contacts.
- **`document-picker.md`** (DocumentPicker) - A library that provides access to the system's UI for selecting documents from the available providers on the user's device.
- **`local-authentication.md`** (LocalAuthentication) - A library that provides functionality for implementing the Fingerprint API (Android) or FaceID and TouchID (iOS) to authenticate the user with a face or fingerprint scan.
- **`mail-composer.md`** (MailComposer) - A library that provides functionality to compose and send emails with the system's specific UI.
- **`notifications.md`** (Notifications) - A library that provides an API to fetch push notification tokens and to present, schedule, receive and respond to notifications.
- **`print.md`** (Print) - A library that provides printing functionality for Android and iOS (AirPrint).
- **`sharing.md`** (Sharing) - A library that provides functionality for sharing and receiving data from other apps.
- **`sms.md`** (SMS) - A library that provides access to the system's UI/app for sending SMS messages.
- **`speech.md`** (Speech) - A library that provides access to text-to-speech functionality.
- **`storereview.md`** (StoreReview) - A library that provides access to native APIs for in-app reviews.
- **`tracking-transparency.md`** (TrackingTransparency) - A library for tracking app users and managing tracking permissions.
