# 📂 router/

Expo Router components and navigation.

## Available Modules

- **`color.md`** (Router Color) - An Expo Router API for accessing platform-specific native colors.
- **`experimental-stack.md`** (Router Experimental Stack) - An opt-in sibling to Stack built on the react-native-screens experimental gamma stack. Available for testing only.
- **`link.md`** (Router Link) - An Expo Router API that provides Link, Redirect, preview, and zoom transition components.
- **`native-tabs.md`** (Router Native tabs) - An Expo Router submodule that provides native tabs layout.
- **`router.md`** (Router) - A file-based routing library for React Native and web applications.
- **`split-view.md`** (Router Split View) - An Expo Router submodule that provides native split view layout.
- **`stack.md`** (Router Stack - Expo Documentation) - Parsed using standard Python library
- **`ui.md`** (Router UI) - An Expo Router submodule that provides headless tab components to create custom tab layouts.
