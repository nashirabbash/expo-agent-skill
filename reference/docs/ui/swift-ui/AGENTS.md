# 📂 swift-ui/

Expo UI components using iOS SwiftUI.

## Available Modules

- **`accessorywidgetbackground.md`** (AccessoryWidgetBackground) - A SwiftUI adaptive background view that provides a standard appearance based on the the widget's environment.
- **`alert.md`** (Alert) - A SwiftUI Alert component for presenting native iOS alert dialogs.
- **`bottomsheet.md`** (BottomSheet) - A SwiftUI BottomSheet component that presents content from the bottom of the screen.
- **`button.md`** (Button) - A SwiftUI Button component for displaying native buttons.
- **`colorpicker.md`** (ColorPicker) - A SwiftUI ColorPicker component for selecting colors.
- **`confirmationdialog.md`** (ConfirmationDialog) - A SwiftUI ConfirmationDialog component for presenting confirmation prompts.
- **`contextmenu.md`** (ContextMenu) - A SwiftUI ContextMenu component for displaying context menus.
- **`controlgroup.md`** (ControlGroup) - A SwiftUI ControlGroup component for grouping interactive controls.
- **`datepicker.md`** (DatePicker) - A SwiftUI DatePicker component for selecting dates and times.
- **`disclosuregroup.md`** (DisclosureGroup) - A SwiftUI DisclosureGroup component for displaying expandable content.
- **`divider.md`** (Divider) - A SwiftUI Divider component for creating visual separators.
- **`form.md`** (Form) - A SwiftUI Form component for collecting user input in a structured layout.
- **`gauge.md`** (Gauge) - A SwiftUI Gauge component for displaying progress with visual indicators.
- **`group.md`** (Group) - A SwiftUI Group component for grouping views without affecting layout.
- **`host.md`** (Host) - A SwiftUI Host component that enables SwiftUI components in React Native.
- **`hstack.md`** (HStack) - A SwiftUI HStack component for horizontal layouts.
- **`image.md`** (Image) - A SwiftUI Image component for displaying SF Symbols.
- **`index.md`** (SwiftUI) - SwiftUI components for building native iOS interfaces with @expo/ui.
- **`label.md`** (Label) - A SwiftUI Label component for displaying text with an icon.
- **`lazyhstack.md`** (LazyHStack) - A SwiftUI LazyHStack component for lazy horizontal layouts.
- **`lazyvstack.md`** (LazyVStack) - A SwiftUI LazyVStack component for lazy vertical layouts.
- **`link.md`** (Link) - A SwiftUI Link component for displaying clickable links.
- **`list.md`** (List) - A SwiftUI List component for displaying scrollable lists of items.
- **`menu.md`** (Menu) - A SwiftUI Menu component for displaying dropdown menus.
- **`modifiers.md`** (Modifiers) - SwiftUI view modifiers for customizing component appearance and behavior.
- **`namespace.md`** (Namespace) - A Namespace component that allows you create Namespaces in SwiftUI
- **`overlay.md`** (Overlay) - A SwiftUI Overlay component for layering content on top of another view.
- **`picker.md`** (Picker) - A SwiftUI Picker component for selecting options from a list.
- **`popover.md`** (Popover) - A SwiftUI Popover component for displaying content in a floating overlay.
- **`progressview.md`** (ProgressView) - A SwiftUI ProgressView component for displaying progress indicators.
- **`rnhostview.md`** (RNHostView) - A component that enables React Native views inside SwiftUI.
- **`scrollview.md`** (ScrollView) - A SwiftUI ScrollView component for scrollable content.
- **`section.md`** (Section) - A SwiftUI Section component for grouping content within lists and forms.
- **`securefield.md`** (SecureField) - A SwiftUI SecureField component for password input.
- **`slider.md`** (Slider) - A SwiftUI Slider component for selecting values from a range.
- **`spacer.md`** (Spacer) - A SwiftUI Spacer component for flexible spacing.
- **`swift-ui.md`** (SwiftUI) - SwiftUI components for building native iOS interfaces with @expo/ui.
- **`swipeactions.md`** (SwipeActions) - A SwiftUI SwipeActions component for adding leading and trailing swipe actions to row content.
- **`tabview.md`** (TabView) - A SwiftUI TabView component for paged or tabbed content.
- **`text.md`** (Text) - A SwiftUI Text component for displaying styled text with support for nested texts.
- **`textfield.md`** (TextField) - A SwiftUI TextField component for text input.
- **`toggle.md`** (Toggle) - A SwiftUI Toggle component for displaying native toggles.
- **`usenativestate.md`** (useNativeState) - A React hook that creates observable state shared between JavaScript and native SwiftUI views.
- **`vstack.md`** (VStack) - A SwiftUI VStack component for vertical layouts.
- **`zstack.md`** (ZStack) - A SwiftUI ZStack component for overlapping layouts.
