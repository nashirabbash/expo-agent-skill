# 📂 drop-in-replacements/

UI components designed as drop-in replacements for core elements.

## Available Modules

- **`bottomsheet.md`** (BottomSheet) - A bottom sheet compatible with @gorhom/bottom-sheet.
- **`drop-in-replacements.md`** (Drop-in replacements) - Components and APIs that serve as drop-in replacements for popular React Native community libraries.
- **`maskedview.md`** (MaskedView) - A masked view compatible with @react-native-masked-view/masked-view.
- **`pagerview.md`** (PagerView) - A horizontally paged view compatible with react-native-pager-view.
- **`segmentedcontrol.md`** (SegmentedControl) - A segmented control compatible with @react-native-segmented-control/segmented-control.
