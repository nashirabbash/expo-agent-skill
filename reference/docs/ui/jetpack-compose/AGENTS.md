# 📂 jetpack-compose/

Expo UI components using Android Jetpack Compose.

## Available Modules

- **`alertdialog.md`** (AlertDialog) - A Jetpack Compose AlertDialog component for displaying native alert dialogs.
- **`badge.md`** (Badge) - A Jetpack Compose Badge component for displaying status indicators and counts.
- **`badgedbox.md`** (BadgedBox) - A Jetpack Compose BadgedBox component for overlaying badges on content.
- **`basicalertdialog.md`** (BasicAlertDialog) - A Jetpack Compose BasicAlertDialog component for displaying dialogs with custom content.
- **`bottomsheet.md`** (ModalBottomSheet) - A Jetpack Compose ModalBottomSheet component that presents content from the bottom of the screen.
- **`box.md`** (Box) - A Jetpack Compose Box component for stacking child elements.
- **`button.md`** (Button) - Jetpack Compose Button components for displaying native Material3 buttons.
- **`card.md`** (Card) - A Jetpack Compose Card component for displaying content in a styled container.
- **`carousel.md`** (Carousel) - Jetpack Compose Carousel components for displaying scrollable collections of items.
- **`checkbox.md`** (Checkbox) - A Jetpack Compose Checkbox component for selection controls.
- **`chip.md`** (Chip) - Jetpack Compose Chip components for displaying compact elements.
- **`colors.md`** (Material Colors) - Read the Material 3 color palette (including Material 3 Dynamic Colors) from JavaScript.
- **`column.md`** (Column) - A Jetpack Compose Column component for placing children vertically.
- **`datetimepicker.md`** (DateTimePicker) - A Jetpack Compose DateTimePicker component for selecting dates and times.
- **`divider.md`** (Divider) - Jetpack Compose Divider components for creating visual separators.
- **`dockedsearchbar.md`** (DockedSearchBar) - A Jetpack Compose DockedSearchBar component for displaying an inline search input.
- **`dropdownmenu.md`** (DropdownMenu) - A Jetpack Compose DropdownMenu component for displaying dropdown menus.
- **`exposeddropdownmenubox.md`** (ExposedDropdownMenuBox) - A Jetpack Compose ExposedDropdownMenuBox component for displaying a dropdown menu with a customizable anchor.
- **`floatingactionbutton.md`** (FloatingActionButton) - Jetpack Compose FloatingActionButton components following Material Design 3.
- **`flowrow.md`** (FlowRow) - A Jetpack Compose FlowRow component for wrapping children horizontally.
- **`horizontalfloatingtoolbar.md`** (HorizontalFloatingToolbar) - A Jetpack Compose HorizontalFloatingToolbar component for displaying a floating action bar.
- **`horizontalpager.md`** (HorizontalPager) - A Jetpack Compose HorizontalPager component for swipeable pages.
- **`host.md`** (Host) - A Jetpack Compose Host component for bridging React Native and Jetpack Compose.
- **`icon.md`** (Icon) - A Jetpack Compose Icon component for displaying icons.
- **`iconbutton.md`** (IconButton) - Jetpack Compose IconButton components for displaying native Material3 icon buttons.
- **`index.md`** (Jetpack Compose) - Jetpack Compose components for building native Android interfaces with @expo/ui.
- **`jetpack-compose.md`** (Jetpack Compose) - Jetpack Compose components for building native Android interfaces with @expo/ui.
- **`lazycolumn.md`** (LazyColumn) - A Jetpack Compose LazyColumn component for displaying scrollable lists.
- **`lazyrow.md`** (LazyRow) - A Jetpack Compose LazyRow component for displaying horizontally scrolling lists.
- **`listitem.md`** (ListItem) - A Jetpack Compose ListItem component for displaying structured list entries.
- **`loadingindicator.md`** (LoadingIndicator) - Jetpack Compose loading indicator components for displaying loading state.
- **`modifiers.md`** (Modifiers) - Jetpack Compose layout modifiers for @expo/ui components.
- **`navigationbar.md`** (NavigationBar) - A Jetpack Compose NavigationBar component for Material 3 bottom navigation.
- **`progress.md`** (Progress Indicators) - Jetpack Compose progress indicator components for displaying operation status.
- **`pulltorefreshbox.md`** (PullToRefreshBox) - A Jetpack Compose PullToRefreshBox component for pull-to-refresh interactions.
- **`radiobutton.md`** (RadioButton) - A Jetpack Compose RadioButton component for single-selection controls.
- **`rnhostview.md`** (RNHostView) - A component that enables React Native views inside Jetpack Compose.
- **`row.md`** (Row) - A Jetpack Compose Row component for placing children horizontally.
- **`searchbar.md`** (SearchBar) - A Jetpack Compose SearchBar component for search input functionality.
- **`segmentedbutton.md`** (SegmentedButton) - Jetpack Compose Segmented Button components for single or multi-choice selection.
- **`shape.md`** (Shape) - A Jetpack Compose Shape component for drawing geometric shapes.
- **`slider.md`** (Slider) - A Jetpack Compose Slider component for selecting values from a range.
- **`snackbar.md`** (Snackbar) - A brief notification that appears at the bottom of the screen to provide feedback without interrupting the user.
- **`spacer.md`** (Spacer) - A Jetpack Compose Spacer component for adding flexible space between elements.
- **`surface.md`** (Surface) - A Jetpack Compose Surface component for styled content containers.
- **`switch.md`** (Switch) - A Jetpack Compose Switch component for toggle controls.
- **`text.md`** (Text) - A Jetpack Compose Text component for displaying styled text.
- **`textfield.md`** (TextField) - Jetpack Compose TextField components for native Material3 text input.
- **`togglebutton.md`** (ToggleButton) - Jetpack Compose ToggleButton components for displaying native Material3 toggle buttons.
- **`tooltip.md`** (Tooltip) - Jetpack Compose Tooltip components for displaying contextual information on long-press.
- **`usenativestate.md`** (useNativeState) - A React hook that creates observable state shared between JavaScript and native Jetpack Compose views.
