# @shopify/flash-list

A React Native component that provides a fast and performant way to render lists.
Android, iOS, tvOS, Web, Included in Expo Go

`@shopify/flash-list` is a "Fast and performant React Native list" component that is a drop-in replacement for React Native's `<FlatList>` component. It "recycles components under the hood to maximize performance."

## Installation

```sh
# npm
npx expo install @shopify/flash-list

# yarn
yarn expo install @shopify/flash-list

# pnpm
pnpm expo install @shopify/flash-list

# bun
bun expo install @shopify/flash-list
```

If you are installing this in an [existing React Native app](/bare/overview.md), make sure to [install `expo`](/bare/installing-expo-modules.md) in your project. Then, follow the [installation instructions](https://shopify.github.io/flash-list/docs/) provided in the library's README or documentation.
