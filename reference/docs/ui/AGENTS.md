# 📂 ui/

UI components and visual capabilities.

## Sub-directories

- **`drop-in-replacements/`**: UI components designed as drop-in replacements for core elements.
- **`jetpack-compose/`**: Expo UI components using Android Jetpack Compose.
- **`swift-ui/`**: Expo UI components using iOS SwiftUI.
- **`universal/`**: Universal UI components for all platforms.


## Available Modules

- **`blur-view.md`** (BlurView) - A React component that blurs everything underneath the view.
- **`bottomsheet_1.md`** (BottomSheet) - A modal sheet that slides up from the bottom of the screen.
- **`button_1.md`** (Button) - A pressable button with multiple visual variants.
- **`checkbox_1.md`** (Checkbox) - A toggle control that represents a checked or unchecked state.
- **`date-time-picker.md`** ('@react-native-community/datetimepicker') - A component that provides access to the system UI for date and time selection.
- **`flash-list.md`** ('@shopify/flash-list') - A React Native component that provides a fast and performant way to render lists.
- **`font.md`** (Font) - A library that allows loading fonts at runtime and using them in React Native components.
- **`gl-view.md`** (GLView) - A library that provides GLView that acts as an OpenGL ES render target and provides GLContext. Useful for rendering 2D and 3D graphics.
- **`glass-effect.md`** (GlassEffect) - React components that render a liquid glass effect using iOS's native UIVisualEffectView.
- **`map-view.md`** (react-native-maps) - A library that provides a Map component that uses Google Maps on Android and Apple Maps or Google Maps on iOS.
- **`masked-view.md`** ('@react-native-masked-view/masked-view') - A library that provides a masked view.
- **`reanimated.md`** (react-native-reanimated) - A library that provides an API that greatly simplifies the process of creating smooth, powerful, and maintainable animations.
- **`safe-area-context.md`** (react-native-safe-area-context) - A library with a flexible API for accessing the device's safe area inset information.
- **`screens.md`** (react-native-screens) - A library that provides native primitives to represent screens for better operating system behavior and screen optimizations.
- **`segmented-control.md`** ('@react-native-segmented-control/segmented-control') - A React Native library that provides a component to render UISegmentedControl from iOS.
- **`skia.md`** ('@shopify/react-native-skia') - A React Native library for creating graphics using Skia.
- **`slider.md`** ('@react-native-community/slider') - A React Native component library that provides access to the system UI for a slider control.
- **`splash-screen.md`** (SplashScreen) - A library that provides access to controlling the visibility behavior of native splash screen.
- **`svg.md`** (react-native-svg) - A library that allows using SVGs in your app.
- **`webview.md`** (react-native-webview) - A library that provides a WebView component.
