# 📂 data-storage/

Modules for local data storage (AsyncStorage, SQLite, SecureStore, etc).

## Available Modules

- **`asset.md`** (Asset) - A universal library that allows downloading assets and using them with other libraries.
- **`async-storage.md`** ('@react-native-async-storage/async-storage') - A library that provides an asynchronous, unencrypted, persistent, key-value storage API.
- **`blob.md`** (Blob) - A web standards-compliant Blob implementation for React Native.
- **`securestore.md`** (SecureStore) - A library that provides a way to encrypt and securely store key-value pairs locally on the device.
- **`sqlite.md`** (SQLite) - A library that provides access to a database that can be queried through a SQLite API.
