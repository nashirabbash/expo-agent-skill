# 📂 sensors/

Device sensors (Accelerometer, Gyroscope, Location, etc).

## Available Modules

- **`accelerometer.md`** (Accelerometer) - A library that provides access to the device's accelerometer sensor.
- **`barometer.md`** (Barometer) - A library that provides access to device's barometer sensor.
- **`devicemotion.md`** (DeviceMotion) - A library that provides access to a device's motion and orientation sensors.
- **`gyroscope.md`** (Gyroscope) - A library that provides access to the device's gyroscope sensor.
- **`location.md`** (Location) - A library that provides access to reading geolocation information, polling current location or subscribing location update events from the device.
- **`magnetometer.md`** (Magnetometer) - A library that provides access to the device's magnetometer sensor.
- **`pedometer.md`** (Pedometer) - A library that provides access to the device's pedometer sensor.
- **`sensors.md`** (Sensors) - A library that provides access to a device's accelerometer, barometer, motion, gyroscope, light, magnetometer, and pedometer.
