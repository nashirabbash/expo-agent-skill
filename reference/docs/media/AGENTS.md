# 📂 media/

Media handling (Audio, Video, Camera, ImagePicker, etc).

## Available Modules

- **`audio.md`** (Audio (expo-audio)) - A library that provides an API to implement audio playback and recording in apps.
- **`camera.md`** (Camera) - A React component that renders a preview for the device's front or back camera.
- **`imagemanipulator.md`** (ImageManipulator) - A library that provides an API for image manipulation on the local file system.
- **`imagepicker.md`** (ImagePicker) - A library that provides access to the system's UI for selecting images and videos from the phone's library or taking a photo with the camera.
- **`live-photo.md`** (LivePhoto) - A library that allows displaying Live Photos on iOS.
- **`media-library-legacy.md`** (MediaLibrary (legacy)) - A library that provides access to the device's media library.
- **`video-thumbnails.md`** (VideoThumbnails) - A library that allows you to generate an image to serve as a thumbnail from a video file.
- **`video.md`** (Video (expo-video)) - A library that provides an API to implement video playback in apps.
